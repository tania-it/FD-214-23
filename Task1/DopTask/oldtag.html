Тождество
Тождеством называют равенство двух тождественно равных выражений.
Два выражения называются тождественно равными, если они принимают одинаковые значения
при всех значениях переменных из их общей области определения.

Одночлен
Одночленом называется произведение чисел, переменных, натуральных степеней переменных.

Стандартный вид одночлена. Коэффициент
Стандартным видом одночлена называется запись одночлена в виде произведения числового множителя,
записанного на первом месте, и степеней переменных с разными основаниями.
Числовой множитель, записанный на первом месте, называется коэффициентом одночлена.

program Avto;
uses GraphABC;
var fon,avto,x,y,w,h:integer; // для описания переменных использовался тег var
begin
SetWindowSize(600,300);
fon:=LoadPicture('gorod.gif');
avto:=LoadPicture('avto.jpg');
SetPictureTransparent(avto,true);
x:=0; y:=170; w:=240; h:=100;
While x < 400 do
begin
ClearWindow;
DrawPicture(fon,0,0);
DrawPicture(avto,x,y,w,h); // для описания переменных использовался тег var
x:=x+10; w:=w-2; h:=h-1;
sleep(20); Redraw;
end;
end.

Тег <kbd> (от англ. keyboard — клавиатура) используется для обозначения текста, который набирается на клавиатуре или для названия клавиш.
Браузеры обычно помечают текст в контейнере <kbd> моноширинным шрифтом.

Теги sup и sub

Тег sub отображает шрифт в виде нижнего индекса: формула серной кислоты: H2SO4

Тег sup отображает шрифт в виде верхнего индекса: упростим одночлен -2a2y2*(-5a3y5)= 10a5y7

Теги <nobr> и <wbr>

1) При использовании HTML тега <nobr> браузер будет отображать текст внутри этого контейнера без переносов строк.
По мере того, как весь текст отображается строкой, появляется горизонтальная полоса прокрутки. Пример ↓↓

Это обычный и очень длинный текст, который весьма неудобно читать, так как он написан в одну строчку, и вы вынуждены прокручивать по горизонтали, чтобы прочитать его.

2) Тег <wbr>указывает браузеру место, где допускается делать перенос строки в тексте, если этого требует ширина родительского элемента. Пример ↓↓
